# Career Bridge

A university application designed to connect students and alumni.

## Prerequisites

Before running the application, please ensure the following software is installed on your system:

1. Java Development Kit (JDK)
2. JavaFX SDK
3. A compatible Java Integrated Development Environment (IDE)

## Installation Guideline

Please follow these instructions to set up and execute the project on your local machine:

1. Extract the contents of the submitted zip archive to a local directory.
2. Open your IDE and import the extracted project folder.
3. Configure the JavaFX library within the project's build path or structure settings. Provide the necessary VM options for JavaFX if required by your specific environment.
4. Build the project to ensure all source files and FXML files compile correctly.
5. Execute the main application class to launch the Career Bridge interface.
