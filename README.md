📘 CareerBridge

CareerBridge is a JavaFX-based desktop application developed as a university project. It aims to connect students and alumni by providing a platform for communication, career guidance, and sharing experiences.

*Features*
👨‍🎓 Student & Alumni Management
💬 Messaging System (stored in messages.txt)
📖 Story Sharing (via stories.txt)
📊 Alumni Career Tracking
📂 File-based data persistence (no database required)
🖥️ JavaFX GUI for interactive user experience

*Technologies Used*
1.Java
2.JavaFX
3.Gradle (Kotlin DSL)

*Installation Guide*
1.Clone the Repository
2.Install Requirements
  Java JDK 11 or higher
  JavaFX SDK installed
  Gradle (optional, wrapper included)
3.You need to add JavaFX to your project.
4.Build the project using : gradlew.bat build
5.Run the launcher class
